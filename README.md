**Motrix** is a lightweight, efficient Entity Component System (ECS) library
designed to integrate seamlessly with **[raylib](https://www.raylib.com)**. It
offers a simple, minimal foundation for building games and interactive
applications following the ECS architecture.

This repository contains both the **Motrix library** itself and a concise example
showcasing a Conway Game of Life implementation using Motrix alongside raylib.

---

## Quick Start

### 1. Clone the repository

```sh
git clone https://github.com/gabrielzschmitz/Motrix.git
cd Motrix 
```

### 2. Build and run

Follow the platform-specific build and run instructions in
[INSTALL.md](INSTALL.md).

## Usage

Motrix serves as a lightweight ECS foundation for your raylib projects. Easily
extend components and systems or build entirely new gameplay on top of it to
suit your needs.
